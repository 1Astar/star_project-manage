# 工作台「调整显示」设计

日期：2026-07-21 · 范围：首页「项目库」卡片区

## 目标

在「今日工作台」项目库区块内嵌「调整显示」，控制**哪些项目出现、以何顺序出现**。

## 非目标

- 不调节星图 / 焦点 / 灵感 / 演进等其它区块
- 不做多用户个人偏好（本版浏览器本地一份配置）
- 不改 `/projects` 完整项目库页

## 行为

1. 项目库标题旁提供「调整显示」
2. 展开面板：非归档项目列表（树形缩进），每项可开关可见；可拖拽排序
3. 即时生效并写入 `localStorage`（`star-pm:workbench-library-v1`）
4. 默认：全部非归档可见，顺序沿用现状（状态优先级 + 父→子树）
5. 新项目未出现在 order 中时：默认可见，排在末尾
6. 去掉原先「只取前 4 个」硬限制，改为按配置展示全部可见项

## 数据

```ts
{ order: string[]; hidden: string[] }
```

- `order`：展示顺序的项目 id
- `hidden`：不显示的项目 id

## UI

- 客户端组件包裹卡片区，避免整页 SSR 依赖 localStorage
- 拖拽用原生 HTML5 DnD（与看板一致，不新增依赖）
